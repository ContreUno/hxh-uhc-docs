# Le Nen

Le **Nen** est l'énergie vitale au cœur du mode de jeu. À l'annonce des rôles, chaque joueur se voit attribuer un **type de Nen** parmi les six existants, indépendamment de son rôle. Le type de Nen offre un **bonus passif léger** qui complète le rôle, sans le remplacer.

Le type de Nen est consultable à n'importe quel moment avec la commande `/hxh nen`.

## Types de Nen

- **Renforcement**
  > Le joueur inflige **+5%** de dégâts au corps à corps.

- **Émission**
  > Le joueur inflige **+5%** de dégâts à distance avec un arc ou un projectile.

- **Manipulation**
  > Les **cooldowns** de capacités du joueur sont réduits de **15%**.

- **Transformation**
  > Le joueur subit **−10%** de dégâts de zone (effets, explosions, AoE).

- **Matérialisation**
  > Le joueur obtient, à l'annonce des rôles, **2 lingots d'or** supplémentaires.

- **Spécialisation**
  > Le joueur obtient, à l'annonce des rôles, un **effet permanent** aléatoire parmi : **Speed I**, **Jump Boost I**, **Night Vision** ou **Water Breathing**.

## Particularités

- Il n'est pas possible de modifier son type de Nen.
- Certains rôles ont des **interactions particulières** avec leur type de Nen :
  > **Gon** est nécessairement de Renforcement.
  > **Kurapika** est nécessairement de Spécialisation.
  > **Hisoka** est nécessairement de Transformation.
  > **Chrollo** est nécessairement de Spécialisation.
  > **Killua** est nécessairement de Transformation.
- Le type de Nen est connu **dès l'annonce des rôles**, le rôle exact peut être différent.
