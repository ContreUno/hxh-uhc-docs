# HxH UHC

Mode de jeu UHC à rôles inspiré de **Hunter × Hunter**.

L'objectif de chaque partie est de gagner avec son camp. À chaque rôle correspondent un ou plusieurs **passifs** ainsi qu'un ou plusieurs **actifs** qui font la particularité du personnage incarné.
